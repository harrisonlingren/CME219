How do I connect to the server/Thomas?
  1. Use your Mac toolbar at the top of your screen and select Go.
  2. Select Connect to Server... .
  3. Enter the following into the server address field:
    smb://thomas.butler.edu
  4. Select Connect.
  5. Enter your Butler username and password.
How do I set up a deployment setting in Aptana?
  1. While in Aptana, click on the brown box icon and select Run Web Deployment Wizard.
  2. The Deploy My Application window will appear. Select FTP/SFTP/FTPS and click Next.
  3. Type in Thomas for the Site Name.
  4. Select SFTP as your Protocol.
  5. Enter the following information in the Remote Info section:
    • Server: thomas.butler.edu
    • Username: your username
    • Password: your user password
    • Remote Path: /home/username/public_html/your folder name
    • Click the Test button to see if the configuration was successful.
  6. In the More Options section, change the Encoding to UTF-8.
  7. Select Finish.
  8. Now to upload your files, you will just select Upload after clicking on the same icon.
How do I create a new project in Aptana?
  1. While in Aptana, click on the New icon and select Web Project. You can also use the Aptana toolbar and select File > New > Web Project.
  2. The New Web Project window will appear. Select Default project and click Next >.
  3. Type in a Project name. (The location should automatically go to the workspace you initially set up in Aptana.
  4. Click the Finish button
  5. Your project will appear in both the Project Explorer and App Explorer.
How do I publish my webpage to the server/Thomas?
  1. In Aptana, click on the brown box icon in the App Explorer toolbar.
  2. Select Upload.
What link do I turn in for assignments?

    You will need to turn in the link to your page on the server.... NOT from the location on your computer. For instance, if you publish a page called Code.html to your Class folder, on the server (Thomas), your URL will look like this:
    http://blue.butler.edu/~username/Class/Code.html

When I view my page in a browser, it says Forbidden. How do I correct this?
  1. While in Aptana, click on the terminal tab in the bottom of your application.
  2. Enter the following text at the prompt:
    ssh username@thomas.butler.edu
  3. You will then be asked for your password. Even though it appears as if you are not typing your password, but you are; the application is password-protecting your password for security reasons.
  4. You should get the following prompt: thomas%
  5. Enter the following text: fix_account
  6. Refresh your browser.
